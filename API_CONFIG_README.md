## 这是什么？

云音乐App播放歌曲时，需要通过一个**音源API**来获取播放链接。

**这个音源API需要你自己准备。** 软件只提供配置入口，你可以填入自己获取到的API地址和密钥。

---

## 配置步骤

### 一步步跟着做：

1. 打开App → 点击右下角 **"我的"**
2. 点击右上角 **设置图标** ⚙️
3. 找到 **"音乐API地址"** → 点击进入

```
┌─────────────────────────────────────┐
│  音乐API配置                         │
│                                     │
│  ① API模式：选择 [本地API]            │
│                                     │
│  ② API地址：                         │
│     [填入你自己的API服务地址]          │
│                                     │
│  ③ 各平台 API Key：                  │
│     QQ音乐 API Key:  [填你的Key]     │
│     网易云 API Key:   [填你的Key]     │
│     酷我音乐 API Key: [填你的Key]     │
│     咪咕音乐 API Key: [填你的Key]     │
│     酷狗音乐 API Key: [填你的Key]     │
│                                     │
│  ④ 点击 [保存]                       │
│  ⑤ 点击 [测试连接] 验证是否成功       │
└─────────────────────────────────────┘
```

4. 如果显示 **"✓ 连接成功"**，配置完成！🎉

## API地址和API Key怎么填

### API地址

- 你需要自己准备一个音源API服务地址
- 地址格式示例：`https://你的域名.com`
- 在App设置中填入即可

### API Key

- API Key 由你的API服务提供方给你
- 不同平台可以填不同的Key
- 如果某个平台不需要Key，留空即可

**API Key 格式要求：**
- 长度：16-128位
- 字符：只能包含字母(a-z A-Z)和数字(0-9)

## QQ音乐 Cookie 配置

> 填写QQ Cookie可以用于获取QQ音乐的播放链接

### 📱 手机获取

1. 手机浏览器打开QQ音乐网页版并登录
2. 登录成功后，在浏览器地址栏输入：
   ```
   javascript:alert(document.cookie)
   ```
3. 弹出的内容就是Cookie，全选复制
4. 粘贴到App设置中的 **"QQ Cookie"** 输入框

### 💻 电脑获取

1. 电脑浏览器打开QQ音乐网页版并登录
2. 按 **F12** 打开开发者工具
3. 切换到 **Network（网络）** 标签
4. 刷新页面（按F5）
5. 点击列表中的**任意一个请求**
6. 右侧找到 **Headers → Request Headers**
7. 找到 **Cookie:** 那一行，复制整串内容
8. 粘贴到App设置中

### ⚠️ 注意：
- Cookie有效期大约 **1-2周**，过期需重新获取
- 必须用**同一个QQ账号**登录

## 音质选择

在设置中选择"默认音质"：

| 音质 | 码率 | 说明 |
|------|------|------|
| 128K | 128kbps | 省流量 |
| 320K | 320kbps | 推荐，均衡 |
| FLAC | 无损 | 最高音质，文件大 |

## 常见问题

### Q: 搜索可以用但播放失败？
**A:** 搜索功能不依赖你的API配置。播放需要正确配置API地址和Key。请检查：
1. 是否已切换到"本地API"模式
2. API地址是否正确填写
3. API Key是否正确
4. 点击"测试连接"验证

### Q: API地址和API Key从哪里获取？
**A:** 你需要自行获取。软件只提供配置入口，不提供API服务。

### Q: 为什么有些歌搜不到？
**A:** 搜索同时查询多个平台，每个平台限时3秒。超时的平台结果不会显示。

### Q: 歌词不显示？
**A:** 不是所有歌曲都有歌词。纯音乐、部分翻唱可能无歌词数据。
