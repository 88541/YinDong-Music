# 需求文档：LX 插件播放器适配

## 简介

确保 LX 插件搜索到的歌曲能在现有播放器中正常播放、切歌、显示进度条和歌词。现有播放器已具备完整的播放控制功能，本需求仅补齐插件模式下的几个小缺口。

## 需求

### 需求 1：插件播放失败自动回退

**用户故事：** 作为用户，我希望插件获取播放链接失败时能自动尝试服务端 API，而不是直接报错。

#### 验收标准

1. IF 插件 musicUrl 请求失败或返回空 URL, THEN THE 播放器 SHALL 自动回退到服务端 MusicApiService.fetchMusicUrl() 获取链接
2. IF 回退也失败, THEN THE 播放器 SHALL 显示"播放失败"错误提示

### 需求 2：进度条时长回退

**用户故事：** 作为用户，我希望进度条始终能正确显示总时长。

#### 验收标准

1. IF ExoPlayer 报告的 duration 为0或负数, THEN THE 播放器 SHALL 使用 Song 模型中的 duration 字段作为显示时长

### 需求 3：插件歌词路由

**用户故事：** 作为用户，我希望播放插件歌曲时歌词能正确加载。

#### 验收标准

1. WHEN 加载插件歌曲歌词时, THE 播放器 SHALL 使用歌曲对应的 source key（而非固定的 lxSelectedSource）请求插件歌词
